# MesaUp

Sistema web de gerenciamento de mesas, comandas, e pedidos em restaurantes.
> Projeto acadêmico construído para demonstrar competências e boas práticas de front-end aplicadas a um cenário de mercado.

## Stack

* Html
* Css
* JavaScript
* Bootstrap
* API Rest

## Principais funcionalidades

* Cadastamento de funcionários do estabelecimento
* Cadastramento de produtos do estabelecimento
* Criação de mesas
* Controle completo das comandas

## Como executar

1. Clone o repositório
2. Abra o projeto no VS Code
3. Execute com Live Server
4. Inicialize a API MesaUp

## Backend

O frontend consome a API MesaUp desenvolvida em Spring Boot.
